Detailed Description

Strict SNOMED definitions for all the below parameters are maintained by Endeavour Predict Using the definitions will ensure the highest possible data quality and consistency being fed to the API Please consult our website for further details on the definitions, which are also available in alternative terminologies. In Attended Mode, clinicians may use their clinical judgement to over-ride automated searches, however this may change the outcome of the risk estimation, and the user assumes all risk for this.
